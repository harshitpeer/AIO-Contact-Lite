AIO Contact Lite is a FREE version of a popular plugin AIO Contact. You can add multiple social media handles in AIO Contact Lite with a user friendly UI.

== Description ==

AIO Contact Lite allows you to add multiple contact options to your website for your users. You can add WhatsApp, Telegram, Skype, Viber, etc. It has a very simply and unique Admin interface which makes it easy to use. You can also sort items with just drag drop by grabbing the item title and moving the item to the desired position. It has a built-in fontawesome icon picker which gives you access to over 1300 different fonts.

== Installation ==

= Option 1 =

* Within your WordPress admin area, navigate to "Plugins".
* Click the "Add New" link near the top of your screen.
* Click the "Upload" link.
* Click the browse button and navigate to wherever you downloaded the zip file to, select the zip file
* Click the "Install Now" button
* Click "Activate Plugin"

= Option 2 =

* Extract the zip file
* Upload (ftp) the resulting `aio-contact-lite` folder to your `/wp-content/plugins/` directory.
* Activate "AIO Contact Lite" through the "Plugins" menu in WordPress
